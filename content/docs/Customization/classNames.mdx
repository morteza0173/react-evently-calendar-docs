# Customizing Styles with `classNames`

You can override default CSS classes using the `classNames` prop:

| Key          | Description                                 |
| ------------ | ------------------------------------------- |
| `root`       | Main container                              |
| `header`     | Calendar header                             |
| `nav`        | Navigation container (month/year + buttons) |
| `month`      | Month text                                  |
| `year`       | Year text                                   |
| `weekdays`   | Weekdays container                          |
| `weekday`    | Each weekday label                          |
| `days`       | Days container                              |
| `day`        | Each day                                    |
| `currentDay` | Highlight for today                         |
| `lastMonth`  | Days from previous month                    |
| `nextMonth`  | Days from next month                        |
