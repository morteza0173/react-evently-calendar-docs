---
title: "renderDay پراپ"
order: 3
---

این پراپ به شما اجازه می‌دهد تا **رندر هر سلول روز در تقویم را سفارشی کنید**.

با استفاده از این پراپ، می‌توانید ظاهر هر روز را به دلخواه خود تغییر دهید، مانند اضافه کردن استایل خاص، آیکون‌ها، یا اطلاعات اضافی داخل روز.

به عنوان مثال

```ts lineNumbers
<Calendar
  renderDay={(day, date, isCurrentMonth, showMonth, isToday, onSelect) => (
    <span
      style={{
        fontWeight: isToday ? "bold" : "normal",
        color: isCurrentMonth ? "black" : "gray",
      }}
      onClick={onSelect}
    >
      {day}
    </span>
  )}
/>
```

پراپ `renderDay` امکان شخصی‌سازی کامل ظاهر هر روز در تقویم را فراهم می‌کند.
این پراپ یک تابع دریافت می‌کند که برای هر روز اطلاعات زیر را در اختیار شما می‌گذارد:

| پراپ             | نوع داده     | توضیح                                                                                                            |
| ---------------- | ------------ | ---------------------------------------------------------------------------------------------------------------- |
| `day`            | `number`     | شماره روز ماه (مثلاً 1، 2، 3 ...)                                                                                |
| `date`           | `Date`       | شی تاریخ جاوااسکریپت مربوط به روز جاری                                                                           |
| `isCurrentMonth` | `boolean`    | آیا این روز متعلق به ماه جاری است یا از ماه قبل/بعد است                                                          |
| `showMonth`      | `boolean`    | آیا باید نام ماه کنار روز نمایش داده شود (معمولاً برای اولین روز ماه یا اگر برای ماه قبل باشد اولین روز ماه قبل) |
| `isToday`        | `boolean`    | آیا این روز امروز است                                                                                            |
| `onSelect`       | `() => void` | تابعی که هنگام انتخاب روز اجرا می‌شود                                                                            |

## نکته‌ی مهم در انتخاب روز

توجه: برای اینکه تغییر انتخاب روز در تقویم ثبت شود، حتماً پراپ `onSelect` را به رویداد `onClick` المنت روزی که می‌سازید وصل کنید. این کار باعث می‌شود که پراپ `onChange` در کامپوننت تقویم اجرا شود.

```ts lineNumbers
<Calendar
  renderDay={(day, date, isCurrentMonth, showMonth, isToday, onSelect) => (
    <span
      className={isToday ? "today" : ""}
      onClick={onSelect} // <- حتما اینجا onSelect رو استفاده کنید // [!code highlight]
    >
      {day}
    </span>
  )}
/>
```
