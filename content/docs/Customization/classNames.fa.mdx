# شخصی سازی استایل ها با استفاده از `ClassNames`

می‌توانید کلاس‌های CSS پیش‌فرض را با استفاده از پراپ classNames بازنویسی کنید.
استایل های پیش فرض را میتوانید از این [لینک](https://github.com/morteza0173/react-evently-calendar/blob/main/src/components/Calendar/Calendar.css) مشاهده کنید

| کلید         | توضیحات                            |
| ------------ | ---------------------------------- |
| `root`       | کانتینر اصلی                       |
| `header`     | هدر تقویم                          |
| `nav`        | کانتینر ناوبری (ماه/سال + دکمه‌ها) |
| `month`      | متن ماه                            |
| `year`       | متن سال                            |
| `weekdays`   | کانتینر روزهای هفته                |
| `weekday`    | هر برچسب روز هفته                  |
| `days`       | کانتینر روزها                      |
| `day`        | هر روز                             |
| `currentDay` | برجسته‌سازی روز جاری               |
| `lastMonth`  | روزهای ماه قبل                     |
| `nextMonth`  | روزهای ماه بعد                     |

توجه داشته باشید که برای بازنویسی (override) استایل‌ها در Tailwind باید از `!` و در CSS از `!important` استفاده کنید.
