---
title: onChange پراپ
---

پراپ `onChange` یک تابع کال‌بک است که وقتی کاربر یک روز را در تقویم انتخاب می‌کند، اجرا می‌شود. این تابع تاریخ انتخاب شده (`Date`) را دریافت می‌کند و می‌توانید از آن برای به‌روزرسانی وضعیت (state) یا ارسال تاریخ به بخش‌های دیگر برنامه استفاده کنید.

### مثال:

```tsx lineNumbers
import React, { useState } from "react";
import Calendar from "react-evently-calendar";

const App = () => {
  const [selectedDate, setSelectedDate] = useState<Date | null>(null);

  return (
    <div>
      <Calendar onChange={(date) => setSelectedDate(date)} />
      {selectedDate && (
        <p>تاریخ انتخاب شده: {selectedDate.toLocaleDateString()}</p>
      )}
    </div>
  );
};

export default App;
```
