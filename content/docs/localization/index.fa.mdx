با استفاده از پراپ `CalendarType` می‌توانید تقویم را بین میلادی (Gregorian) و شمسی (Jalali) تغییر دهید.

### مثال:

```tsx
<Calendar CalendarType="jalali" />  // تقویم شمسی
<Calendar CalendarType="gregorian" />  // تقویم میلادی
```
