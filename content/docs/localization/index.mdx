Using the `CalendarType` prop, you can switch the calendar between **Gregorian** and **Jalali** (Persian) calendars.

### Example

```tsx
<Calendar CalendarType="jalali" />  // Jalali (Persian) calendar
<Calendar CalendarType="gregorian" />  // Gregorian calendar
```
